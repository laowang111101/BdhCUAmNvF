# 大学生期末考核管理系统 java1005

## 项目概述

大学生期末考核管理系统是一款基于Vue和SpringBoot的前后端分离系统。本系统不仅适用于期末考核管理，同时也可方便地被扩展应用于图书管理等领域。

## 技术栈

- **架构：** B/S架构，MVC模式
- **开发环境：** IDEA、JDK1.8、Maven、Mysql
- **编程语言：** Java
- **数据访问：** Mybatis
- **后端框架：** SpringBoot
- **兼容系统：** Windows、Mac

## 功能模块

系统主要功能模块包括：

- 图书信息管理
- 图书类别管理
- 借阅信息管理
- 用户管理

## 系统角色

- 普通用户
- 管理员

## 运行环境

- 前端：Vue.js
- 后端：SpringBoot
- 数据库：Mysql
